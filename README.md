# 程序化材质预设插件 使用手册

> 适用 Blender 5.0+  
> N面板位置：3D 视口→ `N` → **材质预设**

---

## 安装

1. 将 `procedural_material_presets/` 文件夹压缩为 `.zip`
2. Blender → **编辑 → 偏好设置 → 扩展 → 从磁盘安装**
3. 选择 `.zip` 文件并启用

---

## 面板结构

程序化材质预设（主面板）

├─ ① 做旧

## ① 材质生成预设

每个预设折叠面板包含：**参数滑条 + 三个应用按钮**。

### 预设一览

| 预设 | 主要效果 | 参数 |
|------|---------|------|
| 灰尘 | 朝上表面积灰 | 覆盖量、斑块大小 |
| 做旧 | 污垢老化 | 污垢强度、特征尺寸 |
| 划痕 | 金属划痕 | 密度、深度、金属亮度 |
| 污渍 | 水渍/油渍 | 污渍量、大小 |
| 湿润 | 雨后/水迹光泽 | 湿润量、水滴大小 |

### 三种应用模式

| 按钮 | 适用场景 | 行为 |
|------|---------|------|
| **替换** | 全新建材质 | 清空节点树，重建预设 |
| **叠加** | 保留原材质 + 混合 | 在Output 前插入 MixShader |
| **改PBR** | 已有 PBR 贴图的资产 | 直接修改原BSDF 通道（基础色/粗糙度 / 法线 / 金属度） |

> **推荐工作流**：已贴好 PBR 贴图的资产优先使用 **改PBR**，然后叠加 **效果增强**。

### 自定义蒙版（划痕 / 污渍）

- 点击蒙版行右侧 📂 从文件加载黑白图片（自动设置为 Non-Color 色彩空间）
- 白色区域 = 效果完整显示，黑色区域 = 无效果
- 已设蒙版时，"清除蒙版"按钮出现

---

## ③ PBR 材质混合

按通道混合两个材质（基础色 / 粗糙度 / 金属度）。

**使用步骤：**
1. 选择**目标材质**（混合对象的材质B）
2. 可选：加载**混合蒙版**（黑白图片控制区域）
3. 无蒙版时，**混合强度**滑条控制整体比例
4. 点击**执行混合**

> 可多次叠加调用，每次都在当前通道链末端插入 Mix 节点。

---

## ④ 效果增强

在任意预设应用后追加的全局效果，只影响预设节点，不影响已有 PBR。

| 参数 | 说明 |
|------|------|
| 随机种子 | 改变数值让同类物体材质错开，避免克隆感 |
| 重力方向 | 灰尘/污垢根据世界空间法线方向积累（0=关闭，1=完全重力感） |
| 颜色扰动 | 轻微随机化色调，避免纯色死板 |
| 边缘磨损 | 棱角处自动叠加高光金属感（基于 Pointiness） |
| 法线细节 | 叠加高频微细凹凸（不覆盖原法线贴图） |
| 置换强度 | 给材质效果添加真实几何置换（需在Cycles渲染下生效） |
| 蒙版羽化 | 柔化自定义蒙版的边缘过渡（0=硬边，1=完全羽化） |

> **置换说明**：插件会自动启用材质的`Displacement and Bump` 模式，置换强度建议从 `0.005` 开始微调。

## ⑤ 工具与管理

### 基础工具

| 功能 | 说明 |
|------|------|
| 目标材质槽 | 指定应用到物体的第几个材质槽（0=第一槽） |
| 叠加强度 | 叠加模式的 MixShader 混合比例 |
| 复制材质到选中物体 | 将当前活跃物体的材质批量推送给其他选中物体 |
| 封装为节点组 | 将当前材质节点打包为 Node Group（需打开着色器编辑器） |

### 参数配置库

保存和恢复所有滑条数值的命名快照。

1. 在**保存名称**栏输入名称
2. 点击**保存当前参数**
3. 列表中选中配置后，点击**加载**即可恢复

---

##推荐工作流

### 场景一：全新资产
选预设 → 替换 → 调整参数 → 效果增强 → 保存配置



### 场景二：已有 PBR 贴图的资产
改PBR（做旧）→ 改PBR（划痕）→ 效果增强 → 置换强度微调



### 场景三：多材质混合
基础材质已就绪 → PBR材质混合（选材质B +蒙版）→叠加（湿润/灰尘）



### 场景四：批量同类物体
单个物体调好参数 → 保存配置 → 复制材质到选中物体 → 每个物体改随机种子

# xin-TextureTool — KitBash 资产做旧工作流

>适用场景：已导入 KitBash 资产，PBR 贴图已贴好

---

## 核心原则

使用 **改PBR** 模式，不替换原始材质，所有效果叠加在原贴图之上。

---

## 操作步骤

### Step 1 — 做旧

1. 选中资产，打开 N 面板 → **xin-TextureTool**
2. 展开 **① 做旧**
3. 调节参数：
   - 污垢强度：0.3–0.7（越高越脏）
   - 特征尺寸：控制污垢斑块大小
4. 点击 **改PBR**

### Step 2 — 划痕

1. 展开 **① 划痕**
2. 可选：加载自定义黑白蒙版（📂 按钮）
   - 白色区域 = 有划痕，黑色区域 = 无划痕
   - 无蒙版则用划痕密度滑条控制
3. 点击 **改PBR**

### Step 3 — 细节增强

打开 **④ 效果增强**：

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 重力方向 | 0.2–0.5 | 污垢集中在朝上的面 |
| 法线细节 | 0.1–0.2 | 增加微细凹凸质感 |
| 边缘磨损 | 0.1–0.3 | 棱角处自动产生高光磨损 |
| 置换强度 | 0.005–0.02 | 几何置换（仅 Cycles） |

### Step 4 — 批量应用

1. 单个物体效果满意后，多选其他同类物体
2. **⑤ 工具与管理** → 点击 **复制材质到选中物体**
3.逐个物体修改**随机种子**，避免克隆感

---

## 注意事项

- ✅ 始终用 **改PBR**，不用"替换"（替换会清空原贴图）
- ✅ 每步操作后用 `Ctrl+Z` 可撤销
- ✅ 调好参数可在 **⑤ 工具与管理 → 参数配置库** 保存，方便复用
- ⚠️ 置换效果仅在 **Cycles** 渲染器下可见
---

## 注意事项

- **置换效果**仅在 Cycles 渲染器下可见，EEVEE 不支持几何置换
- **改PBR** 要求材质输出端直接连接 Principled BSDF（或通过 MixShader 链接）
- 先用**改PBR**修改通道，再用**叠加**添加整体效果，不要反过来
- 蒙版图片会自动设为 `Non-Color` 色彩空间，无需手动设置
